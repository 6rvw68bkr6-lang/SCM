      📦 자산 변동

      자본 묶임 종합 분석 — 안 팔리는 재고에 자본이 얼마나 묶여 있는가

      코드 단위로 분석 후, 행을 클릭하면 어떤 옵션이 정체 원인인지 옵션 단위 드릴다운으로 확인. "이 옵션만 빼면 코드 회전이 정상" 같은 허수 검증 가능
      자본묶임 지수 계산법
      공식: 재고금액 × (회전일수 ÷ 365일)
      = "이 재고가 1년치로 환산하면 얼마의 자본을 묶어두는가"
      예시: 재고 2,097만원 × (1,707일 ÷ 365) ≈ 9,805만원 (※ 5년 캡 적용 시 6,292만원)
      → 회전이 1년보다 길면 1년치 환산 자본이 재고금액보다 커집니다. 회전이 1년 미만이면 작게 잡힘.
      의미: 단순 우선순위 점수가 아니라, "이 재고를 정리하지 않으면 매년 묶이는 자본 환산값". 회계적으로 기회비용(자본의 시간가치) 개념.

      💰 자본 묶임 TOP — 행 클릭 시 옵션 드릴다운

      실제 재고금액 + 회전일수 + 이익율. 적자 코드의 묶임은 "실제 손실 추정"으로 환산 (자본묶임 × |음수 이익율|)
      #코드품목재고금액회전기간상태이익율자본묶임실제 손실
